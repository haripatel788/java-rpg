# RPG Game (Java)

This repository contains the source code for my text-based RPG game written in **Java**. The game was built using core programming concepts like **loops**, **conditionals**, **user input**, and basic game logic to create an interactive adventure.

## Play the Game
You can play the full game here:

**https://codehs.com/sandbox/id/rpg-Cq6Ldt/run**

## About the Repository
This GitHub repo only includes the **Java source code** for the RPG. All gameplay runs through the CodeHS sandbox linked above.

## Features
- Turn-based combat  
- Player stat tracking  
- Simple enemy AI  
- Branching choices and encounters  
- Replayable structure using loops and conditions  

## Technologies Used
- **Java**  
- Loops 
- Conditionals
- Console input and output  

## How to Run Locally
1. Clone the repo  
2. Open the `.java` file in any Java-compatible IDE or compiler  
3. Run the program  

## Notes
This project was created to practice Java fundamentals and structure a simple game using only core programming tools.
